## ⚙️ 技术栈

- **前端**: Vanilla JavaScript (无框架依赖)
- **API**: Chrome Extension APIs (Manifest V3)
- **存储**: Chrome Storage Sync API
- **UI**: Chrome Side Panel API
- **样式**: CSS3 + 现代化设计
- **图标**: Python PIL 生成

## 🎯 设计原则遵循

1. ✅ **侧边栏面板** - 使用 Chrome Side Panel API
2. ✅ **展示前 3 个框架** - 智能匹配后推荐
3. ✅ **澄清后决策** - 交互式需求完善
4. ✅ **复制提示词** - 一键复制功能
5. ✅ **不保存历史** - 会话数据仅存内存
6. ✅ **资源打包** - 所有框架文件内置
