# 🚀 快速启动指南

## 一分钟上手

### 步骤 1: 加载扩展
1. 打开 Chrome 浏览器
2. 访问 `chrome://extensions/`
3. 开启右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择 `extension` 文件夹
6. ✅ 扩展加载完成!

### 步骤 2: 打开侧边栏
点击浏览器工具栏中的插件图标(紫色 P 图标)

### 步骤 3: 配置 AI 服务

**推荐使用 DeepSeek** (性价比高,速度快)

1. 点击"开始配置"
2. 选择 "DeepSeek"
3. 填写配置:
   - API Key: 您的 DeepSeek API Key
   - 模型: `deepseek-chat`
4. 点击"测试连接"
5. 测试成功后点击"保存配置"

### 步骤 4: 开始使用

在输入框中输入您的需求,例如:

```
我需要为一个在线教育平台设计营销文案,目标用户是25-35岁的职场人士,
希望强调课程的实用性和灵活的学习时间
```

点击发送,系统会:
1. 分析您的需求
2. 推荐 2 个最适合的框架
3. 您选择一个框架
4. 系统提出澄清问题
5. 您回答问题
6. 自动生成优化的提示词
7. 一键复制使用

## 💡 使用技巧

### 1. 描述需求时要清晰
**好的示例**:
```
我需要为一个健身应用写营销文案,目标用户是25-35岁上班族,
希望强调便利性和效果,适合社交媒体投放
```

**不好的示例**:
```
写个文案
```

### 2. 善用澄清问题
系统提出的澄清问题能帮您完善细节,尽量详细回答

### 3. 可以随时调整
生成提示词后,如果不满意可以输入调整需求,如:
```
请增加对价格优势的强调
```

### 4. 选择合适的框架
- **营销文案**: 选择 BAB、SPEAR
- **决策分析**: 选择 RICE、Pros and Cons
- **教育内容**: 选择 Bloom's Taxonomy、ELI5
- **产品设计**: 选择 SCAMPER、HMW

## 🔧 常见问题

### Q: 测试连接失败?
**A**: 
1. 检查 API Key 是否正确
2. 确认网络连接正常
3. 验证模型名称是否准确

### Q: 没有推荐框架?
**A**:
1. 确保输入了足够的需求描述
2. 检查浏览器控制台是否有错误
3. 尝试重新加载扩展

### Q: 生成的提示词不满意?
**A**:
1. 提供更详细的需求描述
2. 认真回答澄清问题
3. 尝试其他推荐的框架
4. 在生成后提出具体的调整需求

### Q: 如何开始新对话?
**A**: 点击右上角的"新对话"按钮

## 🎯 完整流程示例

### 示例: 为健身应用写营销文案

1. **输入需求**:
```
我需要为一个健身应用写营销文案,目标用户是25-35岁的上班族
```

2. **系统推荐**:
```
推荐了 2 个框架:
- BAB Framework (简单)
- SPEAR Framework (中等)  
```

3. **选择框架**: 点击选择 BAB Framework

4. **澄清问题**:
```
系统问: 
1. 您希望强调哪些核心功能?(如便利性、效果、价格等)
2. 文案主要用于哪些渠道?(社交媒体、应用商店等)
```

5. **回答问题**:
```
我输入: 强调便利性和快速见效,主要用于社交媒体投放,
需要有吸引力的开头和明确的行动号召
```

6. **生成结果**:
```
系统基于 BAB 框架生成优化的提示词:

Before (之前): 忙碌的工作让你没时间去健身房?
After (之后): 想象每天只需15分钟,就能在家练出完美身材
Bridge (桥梁): 我们的智能健身应用为上班族量身定制...
[完整提示词]
```

7. **复制使用**: 点击复制按钮,粘贴到您的 AI 助手中使用
